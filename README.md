This application enables users to search for IP addresses and domain names, retrieving their associated geolocation information through an external API service.

🚀 Tech Stack

- Next.js 16
- TailwindCSS
- LeafletJS
- React-Leaflet

## Features

- Search by IP address
- Search by domain
- Display geolocation details

🔧 Environment Variables
This project includes a file named env-example.

1. Rename the file
   env-example → .env
2. Open the .env file and fill in each value based on your own API key and API base URL from your geolocation service provider.

🔧 Installation & Setup

1. git clone https://github.com/rimba17/ip-address-tracker.git

Install dependencies:

npm install

# or

yarn install

Start the development server:
npm run dev

Open in your browser:
http://localhost:3000

🌐 API Source
https://geo.ipify.org/
